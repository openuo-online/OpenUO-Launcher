# OpenUO-Launcher
A launcher for OpenUO  

目前用rust重新开发了一个Launcher启动器 [Another-OpenUO-Launcher](https://github.com/openuo-online/Another-OpenUO-Launcher)，更高的性能、更多的功能、
详情请移步该项目[Another-OpenUO-Launcher](https://github.com/openuo-online/Another-OpenUO-Launcher)，本项目归档不再维护。
  
>非常感谢TazUO的无偿分享，我也会无偿分享开源给大家，只是在TazUO的基础上做了一些汉化；
